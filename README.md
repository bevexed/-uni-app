# 微信小程序

## 框架
项目需在`HBuilder`下运行，打包

### uni-app
- assets:静态文件 => 框架默认 static
- components:Vue 组件 
- plugin:依赖插件
- store:Vuex 
- 相关代码 template:扩展组件
- views:页面 => 框架默认 pages

### UI
